What are Quines?

Jan 26, 2026

I was watching a video the other day by the creator Tsoding about Quines. I highly recommend you watch the video.

But to give you a quick summary, a quine is a program that outputs its own source code as its only output.

The Goal

The question is: How can a program output its own source without being told what that source is?

You need to write a program that contains two things:

  1. String representation of itself
  2. Code that prints a string representation of itself

You can find a bunch of quine programs here, but I’ll be covering Python below.

Simple Quine in Python

The output? The exact same thing!

How it’s Made

  1. The string s contains a template with %r (repr format)
  2. print(s % s) substitutes s into its own template
  3. You get the program’s source code
